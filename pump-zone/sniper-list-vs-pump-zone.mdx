Dequan ships two distinct trading modes, both included in every tier.

## Pump Zone — for discovery

The Pump Zone is the **spatial feed** — the live field where tokens move by market cap and rise by velocity. It is built for **fast pattern recognition** in a real-time market.

Use Pump Zone when you are:

- Hunting fresh launches before the crowd notices
- Watching for inflection points and breakouts
- Holding active positions and want to see them living
- Working in 30-second decision cycles

The Pump Zone surfaces a **Quick Buy** preset on every token — one tap from the candle drawer fires your default bet size — because reaction time is the whole point.

## Sniper List — for analysis

The Sniper List is the **analyst view** — a sortable, filterable table with charts inline and full detail panels. It is built for **deliberate research** when you have time to look.

Use Sniper List when you are:

- Doing due diligence on a single token across multiple metrics
- Comparing several tokens side by side
- Drawing on a chart, setting alerts, working out a thesis
- Working in 5-minute or longer decision cycles

The Sniper List shows the same live data as the Pump Zone — same fan-out pipeline, same metrics, same safety signals — just rendered as a table you can sort, filter, and study.

Most experienced Dequan users keep Pump Zone open as their primary surface and **dip into Sniper List** when a specific token earns deeper attention. The candle drawer on every Pump Zone token has an `Open in Sniper List` shortcut for exactly this reason.

## What lives only in one mode

A small number of features are mode-specific by design:

| Feature | Pump Zone | Sniper List |
|---|---|---|
| Spatial field rendering | ✅ | — |
| Velocity bands (filter) | ✅ | ✅ (as column filters) |
| Live candle drawer | ✅ | ✅ |
| Inline micro-history | ✅ | ✅ |
| Quick Buy preset (default ON) | ✅ | ✅ (opt-in) |
| Full chart drawer | ✅ | ✅ |
| Full Token Detail Tabs | ✅ | ✅ |
| Watchlist column | — | ✅ |
| Multi-column sort | — | ✅ |
| Saved filter sets | — | ✅ |

## Switching modes

A toggle at the top of the app moves you between modes instantly. State is preserved — your active position, your watchlist, your filters — so the switch is a context shift, not a reload.

You can also pop a chart out of either mode into a floating window and keep it onscreen while you do something else. The chart popout is a fully featured surface — it includes Quick Buy, the live PnL, milestones, and one-tap sells.

<Note>
**The data does not differ between modes.** Same upstream pipeline, same metrics, same updates per second. The mode is purely a presentation choice.
</Note>
