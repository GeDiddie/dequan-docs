## Every feature is in the free tier.

There are three Dequan tiers — **Scout**, **Sniper**, and **Apex**. Sniper and Apex are invite-only.

Read this carefully: **the entire feature set is available on Scout, the free tier.** Pump Zone, Lightning Wallet, honeypot guard, drain detection, racing lanes, charts, every safety system, every signal — all of it. The paid tiers raise limits (more concurrent watches, higher throughput, lower fees), they do not unlock features.

We don't believe in a paywalled trading experience. Either the platform protects every user equally, or it isn't a real platform.

## Built for serious traders. Engineered like infrastructure.

Dequan runs on dedicated hardware in a single low-latency colocation, behind Cloudflare's global edge for reach. Every layer — discovery, analysis, execution, settlement — is built in-house and instrumented end-to-end. We publish our own latency. We surface our own health. When a node is degraded, you see it before it affects you.

This is not a Telegram bot in a trench coat. It is enterprise-grade software that happens to feel like a video game on purpose.
